# Saeduck e ne, 세상의 모든 덕질   
   
 세상에는 많은 사람들과 그 수만큼 많은 관심사들이 있습니다.  
 
 [Saeduck e ne(a.k.a. 세덕이네)](http://sussaeduck.store/)는 세상의 모든 관심사들을 모을 수 있습니다.   
 세덕이네로 놀러와 다양한 관심사를 나눠보세요!
